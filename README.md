Good accessibility is not just a technical requirement — it is a responsibility we have as developers to ensure the web remains open and welcoming to all.

I approach development with care, patience, and attention to detail. Whether I’m writing code, designing a layout, or solving a problem, I aim to create work that is dependable, maintainable, and built with integrity.
